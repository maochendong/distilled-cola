## 战略总纲：服务差异化

### 核心理念

整个行业以房源为中心，按面积/价格粗筛匹配；蒸馏小可乐要做的事是 **反过来的**——以人为中心，按购房者的身份和人生阶段提供差异化的决策支持。

### 四大垂直服务模块

| 模块 | 目标人群 | 核心关切 |
|------|----------|----------|
| **FirstHome 🏠** | 首套房年轻人刚需上车（25-35 岁，第一套房，预算有限） | 通勤便利、总价门槛、首付月供、增值潜力、「先上车」时机 |
| **HerHome 🏡** | 独立女性购房者（单身/离异/不婚/有积蓄寻求安全感的女性） | 安全底线、产权规划、长期流动性 |
| **FamilyUpgrade 👨‍👩‍👧‍👦** | 中年有娃家庭改善（30-45 岁，二胎/学龄儿童，卖旧买新） | 学区、空间动线、置换链条、社区儿童友好度、通勤双平衡 |
| **GoldenYears 🌅** | 银发族养老置业（60+ 老人自住/子女代购） | 医疗保障、无障碍通道、社区支持系统 |

### 覆盖完整生命周期

```
         FirstHome                FamilyUpgrade             GoldenYears
    🔰 年轻人刚需上车  →  👨‍👩‍👧‍👦 家庭改善置换  →  🌅 养老安度晚年
          ↑                               ↑
          └──── HerHome ────┘
             独立女性购房（贯穿全生命周期的一条独立主线）
```

### 落地路径

```
V1.x 当前：通用问答 → 所有人得到相同答案
 ↓
V2.0 用户画像 + 身份路由 → 系统知道「在为谁服务」
 ↓
V2.5 四大模块专题知识 + 评分器 → 差异化评估
 ↓
V3.0 AI 选房顾问 + B2B 赋能 → 平台级能力
```

> 所有差异化功能以 **[FirstHome]** / **[HerHome]** / **[FamilyUpgrade]** / **[GoldenYears]** / **[通用]** 标签标注所属模块。
